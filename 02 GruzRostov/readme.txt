Сайт по грузоперевозкам GruzRostov.
Выполнен в формате Landing page.
Использовал фреймворк Laravel.
В нем реализованы: 
- слайдеры (в первой секции, в секциях "Услуги" и "Наши клиенты");
- кальултяор расчета стоимости услуги;
- окно для отправки заявки на обратный звонок;
- адаптивность под планшеты и телефоны, разной ширины (до 320px);
- якори по странице;
- админка с возможностью редактирования/удаления/добавления: информации на сайте, картинок, слайдеров, сотрудников;
- страница оператора, для обрабатывания заявок на обратный звонок.
